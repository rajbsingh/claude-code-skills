# Humanizer Skill

## Purpose

Rewrite text so it sounds more natural, professional, and human.

## Use this skill when

- Text sounds too AI-generated
- Text feels too formal or robotic
- The user wants a human tone
- The user wants LinkedIn, email, book, or website copy improved

## Instructions

1. Preserve the original meaning.
2. Make the language natural and clear.
3. Avoid exaggerated marketing language.
4. Avoid phrases that sound like ChatGPT.
5. Keep the user's intent and voice.
6. Improve flow, grammar, and readability.

## Output format

Return only the improved version unless the user asks for explanation.
