# AI Book Outline Builder Skill

## Purpose

Help create strong, structured, reader-friendly AI book outlines for professionals, leaders, students, founders, and general readers.

## Use this skill when

- The user wants to write an AI book
- The user needs chapter structure
- The user wants a bestselling-style book flow
- The user wants a 6x9 paperback/hardcover/eBook plan
- The user wants executive-level AI content
- The user wants a human-sounding book structure

## Instructions

1. Identify the target reader.
2. Clarify the promise of the book.
3. Build a logical chapter sequence.
4. Avoid repetitive chapter titles.
5. Use clear section headings.
6. Balance:
   - explanation
   - examples
   - business value
   - practical application
   - future outlook
7. Keep the flow human and natural.
8. Avoid making the book sound like a blog post collection.
9. Suggest front matter and back matter where useful.

## Output format

Use this structure:

## Book Positioning
Explain who the book is for and what it promises.

## Recommended Title
Suggest title if needed.

## Subtitle Options
Suggest subtitle options if needed.

## Full Chapter Outline
Give chapters in sequence.

## Chapter Breakdown
For each chapter, include:
- Purpose
- Key sections
- Reader takeaway

## Front Matter
Suggest opening sections.

## Back Matter
Suggest closing sections.

## Final Recommendation
Give practical publishing guidance.
