# Decision Framer Skill

## Purpose

Help the user make a clear decision by framing the situation, comparing options, and identifying the best practical next step.

## Use this skill when

- The user is confused between multiple options
- The user asks “which is better?”
- The user needs a founder, business, career, or product decision
- The user wants realistic pros and cons
- The user needs a practical recommendation, not generic advice

## Instructions

1. First, restate the decision clearly.
2. Identify the main options.
3. Compare options using practical criteria:
   - Cost
   - Time
   - Risk
   - Speed
   - Long-term value
   - Revenue potential
   - Execution difficulty
4. Mention hidden risks or assumptions.
5. Give a direct recommendation.
6. End with the next 1–3 actions the user should take.

## Output format

Use this structure:

## Decision
State the decision in one line.

## Options
List the available options.

## Comparison
Compare the options clearly.

## Recommendation
Give one clear recommendation.

## Next Steps
Give simple action steps.
