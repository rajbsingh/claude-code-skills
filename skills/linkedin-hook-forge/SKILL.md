# LinkedIn Hook Forge Skill

## Purpose

Create strong LinkedIn hooks that make people stop scrolling and continue reading.

## Use this skill when

- The user wants a LinkedIn post
- The user wants better opening lines
- The user wants content ideas
- The user wants founder, AI, startup, career, or business posts
- The user wants human-sounding, non-generic LinkedIn writing

## Instructions

1. Understand the target audience.
2. Identify the emotional angle:
   - curiosity
   - pain
   - mistake
   - insight
   - transformation
   - contrarian view
3. Create hooks that are short, direct, and human.
4. Avoid fake hype.
5. Avoid overused phrases like:
   - “In today’s fast-paced world”
   - “AI is changing everything”
   - “Here’s what nobody tells you”
6. Make the opening line strong but believable.
7. Provide multiple hook options when useful.

## Output format

Use this structure:

## Best Hooks

1. Hook option one
2. Hook option two
3. Hook option three
4. Hook option four
5. Hook option five

## Recommended Hook

Pick the strongest one and explain briefly why.

## Optional Post Direction

Suggest the best angle for the full post.
